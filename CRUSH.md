
# Crush Quickstart

This file contains commands and guidelines for working in this repository.

## Commands

- **Build:** `npm run build`
- **Lint:** `npm run lint`
- **Test:** `npm test`
- **Test a specific file:** `npm test -- <file_path>`

## Code Style

- **Formatting:** This project uses Prettier for code formatting. Run `npm run lint` to format your code.
- **Imports:** Use ES module imports.
- **Naming Conventions:** Use camelCase for variables and functions. Use PascalCase for classes.
- **Error Handling:** Use try/catch blocks for handling errors.
